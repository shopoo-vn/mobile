# Mobile — Marketplace (React Native)

Repo `mobile/` trong workspace `shopoo/`. App người dùng cuối: duyệt/đăng tin,
chat real-time, nhận push. Gọi **Auth / Listing / Media / Chat** qua REST +
WebSocket (Socket.io) + FCM.

> ⬜ Chưa khởi tạo. Kế hoạch chi tiết: [`../implementation_plan_frontend.md`](../implementation_plan_frontend.md) (mục **1. MOBILE APP**).

Dự kiến: Expo + React Navigation, TanStack Query, Zustand, socket.io-client,
expo-notifications (FCM). Khởi tạo: `npx create-expo-app@latest .`
